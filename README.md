The Amandla Highschool Locker Booking system will be an online website-based system where both parents and users can book and manage lockers for students.

Purpose and user requirements for Parent users:
-	Parents to be able to create user account, they will be able access using authentication. 
-	Parents will be able to create students under their parent profile.
-	 Parents will be able to book a locker for each student. 
-	The system will automatically add parents to a waiting list and automatically inform the school of this. 
-	The parent will be able to track the status of their locker booking.
-	The parent will automatically be informed once payment is received.
-	Parents will be informed if there is an available locker and that they have been officially allocated a locker for their respective children/students. 
-	Parents will also be able to cancel bookings. 

Purpose and user requirements for School Staff Administrator users:
-	Admin to log into an admin account securely. 
-	Admin to add parent accounts to assist users.
-	Admin to be able to add students to parent accounts for users. 
-	Administrators able to view waiting lists. 
-	Admin to approve payments received. 
-	Admin able to view available lockers. 
-	Admin to officially allocate students to lockers that are available. 
-	Administrators informed if there are no lockers available. 
-	Administrators will be able to also cancel locker bookings that remain on the waiting list.  
-	Administrators able to see various management information system reports to help manage locker bookings. 
-	Admin MIS reports will provide the administrators, and school managers with information regarding lockers already booked, client information regarding bookings, amounts of lockers remaining, and statistical information to aid in decision making and analytics for the school.
